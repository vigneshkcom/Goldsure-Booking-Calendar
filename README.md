# Goldsure Booking Calendar

Public booking page for **booking.goldsure.com.au**, hosted on Vercel.

- Single static page (`index.html`) — no build step.
- Gold-on-black design matching the Goldsure brand, with an animated hero.
- Embeds the LeadConnector booking widget
  (`https://api.leadconnectorhq.com/widget/booking/udRwQGz9i7JY4zRYRQyc`)
  and its `form_embed.js` auto-resize script.
- If the widget can't load, an in-brand fallback calendar is shown.

## Deploy (Vercel)
1. Import this repository into Vercel (Framework preset: **Other**, no build command).
2. Add the domain `booking.goldsure.com.au` in the project's Domains settings.
